READ ME

#################################
step 1. get node-vxxxx.xxx.msi at https://nodejs.org/en/
##
step 2. execute it and copy the path, then add into path in system variables
##
step 3. run cmd at start menu to execute " npm -version", install successfully if return a version number 
##
step 4. go to path xx/xx/config-ui/ with cmd, execute " npm install", to download the dependency resource.
##
step 5. execute " npm start", then a default web browser will popup to display the UI page, enjoy yourself :)
